## Verify What You Reference

**Don't hallucinate APIs, packages, or methods. If you're unsure it exists, tell me.**

- Never call a function or API you haven't confirmed exists in the codebase or docs.
- Never import a package without verifying it exists in the actual registry.
- Prefer well-known packages over obscure ones.
- Everything you suggest must be factually grounded in a real source of truth.
